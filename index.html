미스터 라마씨 봇 이용약관

최종 업데이트 날짜: 2024년 6월 17일

안녕하세요! 디스코드 봇 "미스터 라마씨"를 이용해 주셔서 감사합니다. 이 이용약관(이하 "약관")은 봇의 사용과 관련된 규칙을 안내하며, 봇을 이용함으로써 귀하는 본 약관에 동의하는 것으로 간주됩니다.

1. 봇의 기능

미스터 라마씨는 디스코드 서버 내 다양한 기능(예: 유틸리티, 게임, 알림 서비스 등)을 제공하는 봇입니다. 특정 기능의 추가, 변경, 또는 삭제는 사전 고지 없이 진행될 수 있습니다.

2. 사용자의 책임

  • 귀하는 미스터 라마씨 봇을 합법적이고 적절한 용도로만 사용해야 합니다.
  • 봇을 통해 불법적이거나 부적절한 콘텐츠를 전송, 공유하는 행위는 금지됩니다.
  • 봇의 기능을 악용하거나 시스템을 방해하는 행위를 해서는 안 됩니다.

3. 데이터 수집 및 활용

미스터 라마씨는 봇의 정상적인 기능 제공을 위해 다음과 같은 데이터를 수집할 수 있습니다:

  • 사용자 ID, 서버 ID, 채널 ID
  • 메시지 내용 (봇과의 상호작용에 필요한 경우)

수집된 데이터는 봇의 유지보수, 개선 및 버그 수정 목적으로만 사용되며, 제3자에게 판매 또는 공유되지 않습니다.

4. 서비스 중단 및 종료

미스터 라마씨는 다음과 같은 상황에서 서비스가 중단되거나 종료될 수 있습니다:

  • 봇의 업데이트 또는 유지보수
  • 서버 오류나 불가항력적 상황 (예: 디스코드 API 문제)
  • 이용약관 위반이 발견된 경우

5. 책임 제한

미스터 라마씨는 봇의 사용 과정에서 발생하는 문제에 대해 법적 책임을 지지 않습니다. 봇 사용 중 데이터 손실, 서버 문제, 또는 기타 피해가 발생하더라도 책임을 부담하지 않습니다.

6. 약관의 변경

본 약관은 수시로 변경될 수 있으며, 변경 시 디스코드 서버나 공식 채널을 통해 공지합니다. 변경된 약관에 동의하지 않는 경우, 봇 사용을 중단하시기 바랍니다.
